## 🌌 代表性项目

### 🎶 [ACE-script](https://github.com/yjzxkxdn/ACE-script)
一套面向 **ACE Studio** 的 Python 自动化脚本库，让音符编辑、轨道控制变得像写代码一样流畅。  
> “它是 ACE MCP 服务的 Python wrapper，为自动化创作打开新可能。”

### 🔀 [HarmonicNoiseSplitTrainingSet](https://github.com/yjzxkxdn/HarmonicNoiseSplitTrainingSet)
无需训练 DDSP，自动生成用于**气声谐波分离**的高质量数据集。  
> 为干净的歌声合成铺路。

### 🧪 [Mini-DDSP](https://github.com/yjzxkxdn/Mini-DDSP)
轻量级可训练 DDSP 声码器实现，适合快速实验与教学。  
> 已获 15+ Stars，欢迎试用！

### 🎤 [hifisampler / hachimisampler](https://github.com/yjzxkxdn/hifisampler)
基于 `pc-nsf-hifigan` 的 UTAU 高质量重采样器，让虚拟歌者的声音更自然。  
> “胖宝宝”也值得被温柔对待 ❤️
